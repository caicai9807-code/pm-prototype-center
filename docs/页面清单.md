# 页面清单

用于维护原型仓库中所有业务模块、页面入口、跳转关系和页面职责。

## 建议填写规则

- 一行代表一个页面。
- 页面名称尽量与最终 HTML 文件名保持一致。
- 入口页面、关键操作页、结果页、异常页都应纳入清单。
- 若页面存在多个状态，建议拆分记录或在备注中写清。

## 页面跳转关系图

```
index.html（模块首页）
  └── task-list.html（面积检查任务管理）
        ├── [+ 新建任务] → task-create.html（步骤1：基础信息与站点选择）
        │     └── [下一步] → task-expert.html（步骤2：抽选专家与分组分配）
        │           ├── [上一步] → task-create.html
        │           └── [提交任务] → task-list.html
        ├── [详情] → task-detail.html（任务详情）
        │     └── [返回] → task-list.html
        └── [编辑] → task-detail.html?mode=edit
```
