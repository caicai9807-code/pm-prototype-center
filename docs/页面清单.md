# 页面清单

用于维护原型仓库中所有业务模块、页面入口、跳转关系和页面职责。

## 建议填写规则

- 一行代表一个页面。
- 页面名称尽量与最终 HTML 文件名保持一致。
- 入口页面、关键操作页、结果页、异常页都应纳入清单。
- 若页面存在多个状态，建议拆分记录或在备注中写清。
