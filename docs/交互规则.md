# 交互规则

用于定义页面中的校验、联动、状态切换、权限差异和异常提示规则。

## 建议填写规则

- 优先记录会影响演示效果和业务理解的关键交互。
- 一个规则描述一个明确场景，避免混写。
- 对按钮可用性、字段显隐、审批动作、批量操作要写清前置条件。
- 若规则依赖角色或状态，必须显式写出。

### 8.3 误差判断规则

| 实测建筑面积范围 | 判断标准 | 计算公式 | 超限判定 |
| --- | --- | --- | --- |
| ≤500㎡ | 两次测量相对误差不超过 ±3% | `\|复核面积 - 普查面积\| / 普查面积 × 100% > 3%` | 超出即超限 |
| 500㎡ < 面积 ≤ 2000㎡ | 两次测量相对误差不超过 ±2% | `\|复核面积 - 普查面积\| / 普查面积 × 100% > 2%` | 超出即超限 |
| >2000㎡ | 两次测量绝对误差不超过 ±60㎡ | `\|复核面积 - 普查面积\| > 60` | 超出即超限 |
