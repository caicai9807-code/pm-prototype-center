# 字段字典

用于沉淀原型中的字段定义、展示规则、录入规则和业务口径。

## 建议填写规则

- 先记录页面核心字段，再补齐扩展字段。
- 展示字段和录入字段都要纳入。
- 状态、枚举、金额、时间、人员、组织类字段优先写清口径。
- 若字段在多个页面复用，建议统一命名并保持口径一致。
