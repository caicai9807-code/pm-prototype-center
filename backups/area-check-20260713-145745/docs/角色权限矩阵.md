# 角色权限矩阵

| 角色 | 页面/模块 | 可执行操作 | 数据范围 | 状态限制 | 备注 |
| --- | --- | --- | --- | --- | --- |
| **业务管理员** | 面积检查任务管理 | 查看任务列表、筛选、排序 | 全部任务 | 全部状态 | 任务管理主入口 |
| | 新建/编辑任务 | 新建任务、编辑草稿/待开始任务（基础信息+站点+专家） | 全部任务 | 草稿/待开始 | 进行中任务仅可改专家分配 |
| | 编辑任务（进行中） | 编辑专家分配 | 全部任务 | 进行中 | 基础信息+站点锁定 |
| | 查看任务详情 | 查看任务信息、进度、专家分配 | 全部任务 | 全部状态 | 只读 |
| | 审核整改通知书 | 审核通过/驳回 | 本部门/全部门 | 待审核 | ≤30工作日自动通过，>30日人工 |
| | 查看整改任务 | 查看整改任务列表和详情 | 全部 | 全部状态 | 只读，不操作整改 |
| | 查看站点明细 | 查看站点信息和导出 | 全部站点 | 全部状态 | — |
| **检查专家** | 我的面积检查任务 | 查看任务列表、筛选 | 本人负责的任务 | 全部状态 | 含待确认/待开始/进行中/已结束 |
| | 确认任务 | 确认接受任务 | 本人 | 待确认 | 确认后进入待开始 |
| | 拒绝任务 | 填写拒绝原因后拒绝 | 本人 | 待确认 | 拒绝后进入转派流程 |
| | 开始检查 | 进入检查页面 | 本人 | 待开始/进行中 | — |
| | 查看普查信息 | 查看站点面积、明细、图纸 | 本人负责的站点 | 全部状态 | 只读 |
| | 填写检查记录单 | 填写7项检查项、提交/暂存 | 本人负责的站点 | 检查中/待修改 | 争抢式填写，先点先得 |
| | 生成整改通知单 | 填写整改通知单内容 | 本人负责的站点 | 不合格 | 双人确认后生效 |
| | 实测复核 | 上传复核照片、填写复核面积 | 本人负责的站点 | 仅普查依据为"实测"的建筑 | 失焦自动保存 |
| | 检查记录单确认 | 同意/驳回检查记录单 | 本人参与的任务 | 确认中 | 填写人默认已同意 |
| | 整改通知书确认 | 同意/驳回整改通知书 | 本人参与的任务 | 待生效 | 双人同意后进入审核 |
| | 整改复查确认 | 同意/驳回整改完成 | 本人参与的任务 | 待专家审核 | 全部同意→已完成 |
| **部门领导** | 转派审核列表 | 查看转派任务（全部/待处理/已处理） | 本部门 | 全部 | 仅当专家拒绝时出现 |
| | 审核指派 | 查看任务摘要、原专家拒绝原因、转派日志、选择新专家 | 本部门内 | 待处理 | 确认指派后新专家默认已确认 |
| | 查看转派记录 | 查看已完成转派的时间线 | 本部门 | 已处理 | — |
| **整改责任人** | 整改任务列表 | 查看待整改/整改中/已超期/已完成任务 | 本人负责 | 全部状态 | 按片区所所长自动分配 |
| | 接收整改任务 | 查看整改截止时间，确认接收 | 本人 | 待整改 | 接收后计时开始 |
| | 提交整改 | 填写整改说明（500字）、上传整改告知书附件 | 本人 | 整改中/已超期 | 整改说明必填，附件必传 |
| | 查看整改详情 | 查看整改要求、整改说明、确认状态 | 本人 | 全部状态 | 只读查看 |
| **管理部主任** | 查看本管理部整改/检查情况 | 查看本管理部站点检查状态和整改状态 | 本管理部 | 全部状态 | 仅查看，不参与操作 |
| | 查看统计汇总 | 查看站点明细汇总 | 本管理部 | 全部状态 | — |
| | 处理检查录入 | — | — | — | 不直接处理专家检查录入 |
